RUN THE WEB APPLICATION

To run the application first activate the virtual environment from the django_env folder
- In command prompt, CD to the django_env folder location
- type .\django_env\Scripts\activate in the command line to activate the virtual envirment

Once in the virtual environment
- In command prompt window, CD to the project folder
- type python manage.py runserver
- open web browser
- In address bar type in localhost address
- Web application should appear in the browser
